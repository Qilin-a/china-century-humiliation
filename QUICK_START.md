# ⚡ 快速开始指南

## 🎯 5分钟上手新功能

### 第一步：打开网站
直接打开 `index.html` 文件

### 第二步：体验核心功能

#### 1️⃣ 切换主题（5秒）
- 点击导航栏右侧的 🌙 图标
- 观察页面从深色变为浅色
- 再次点击切换回深色

#### 2️⃣ 使用搜索（10秒）
- 按下 `Ctrl + K`（Mac用 `Cmd + K`）
- 输入"鸦片"或"战争"
- 点击搜索结果自动跳转
- 按 `ESC` 关闭搜索

#### 3️⃣ 收藏事件（15秒）
- 滚动到"历史脉络"区域
- 点击任意事件右上角的 ☆ 图标
- 点击右侧浮动的 ★ 按钮查看收藏
- 点击收藏项跳转到对应事件

#### 4️⃣ 移动端菜单（仅移动设备）
- 调整浏览器窗口到手机大小
- 点击右上角的三条横线
- 体验滑动菜单

#### 5️⃣ 返回顶部
- 向下滚动页面
- 点击右下角的 ↑ 按钮
- 平滑滚动回顶部

---

## 🎮 快捷键速查

| 快捷键 | 功能 |
|-------|------|
| `Ctrl/Cmd + K` | 打开搜索 |
| `ESC` | 关闭弹窗 |
| `Ctrl/Cmd + /` | 移动菜单（移动端）|

---

## 💡 使用技巧

### 搜索技巧
- 输入年份：如"1900"
- 输入关键词：如"条约"、"战争"
- 输入地点：如"南京"、"北京"

### 收藏技巧
- 收藏感兴趣的事件方便复习
- 收藏数据自动保存，不会丢失
- 可以随时清空所有收藏

### 主题技巧
- 深色主题适合夜间阅读
- 浅色主题适合白天阅读
- 主题偏好自动记忆

## 📱 移动端特别说明

在手机或平板上：
1. 导航菜单自动隐藏
2. 点击汉堡图标打开菜单
3. 语言切换在移动菜单中
4. 所有功能完全可用
