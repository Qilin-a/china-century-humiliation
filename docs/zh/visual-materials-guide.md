# 视觉材料指南

本指南为中国百年国耻历史文档项目提供添加历史图片、地图、照片和其他视觉材料的建议。

---

## 概述

视觉材料可以显著增强对历史事件的理解。本指南概述：
- 包含的视觉材料类型
- 版权和许可注意事项
- 推荐来源
- 文件组织和命名规范
- 署名要求

---

## 版权和许可

### 公有领域材料

**通常可安全使用**：
- 1928年前出版的作品（在大多数司法管辖区）
- 中国政府文件（查看具体法律）
- 作者去世70年以上的作品
- 明确标记为公有领域的作品

**使用前验证**：
- 检查相关司法管辖区的版权状态
- 确认出版日期
- 如适用，验证作者去世日期

### 知识共享材料

**可接受的许可**：
- CC0（公有领域）
- CC BY（署名）
- CC BY-SA（署名-相同方式共享）- 与我们的许可兼容

**避免**：
- CC BY-NC（署名-非商业性使用）
- CC BY-ND（署名-禁止演绎）
- 保留所有权利

### 合理使用考虑

**教育用途**：
- 本项目符合教育目的
- 仅使用必要内容
- 提供适当署名
- 优先转换性使用

**最佳实践**：
- 尽可能寻求许可
- 优先使用公有领域材料
- 始终提供清晰署名

### 命名规范

**格式**：`描述-年份.扩展名`

**示例**：
- `lin-zexu-portrait.jpg`
- `treaty-nanking-1842.jpg`
- `battle-of-yellow-sea-1894.png`
- `may-fourth-protest-1919.jpg`

**规则**：
- 使用小写字母
- 用连字符分隔单词
- 相关时包含年份
- 描述性但简洁
- 使用标准文件扩展名

## 图像规格

### 技术要求

**分辨率**：
- 最小：最长边1200像素
- 首选：2000像素以上用于存档质量
- 地图：更高分辨率以显示细节

**文件格式**：
- 照片：JPEG（质量85-95%）
- 图形：PNG用于透明度
- 示意图：尽可能使用SVG
- 文件：高分辨率JPEG或PDF

**文件大小**：
- 平衡质量和文件大小
- 优化网页显示
- 保留原件用于存档

### 可访问性

**替代文本**：
- 提供描述性替代文本
- 解释显示内容
- 包含相关背景
- 保持简洁但信息丰富

**说明文字**：
- 日期和地点
- 简要描述
- 历史背景
- 来源署名

## 署名要求

### 必需信息

每张图片提供：

1. **标题/描述**：图片显示内容
2. **日期**：创建/拍摄时间
3. **创作者**：摄影师、艺术家或组织
4. **来源**：获取地点
5. **许可**：版权状态或许可类型
6. **网址**：如在线则链接到原始来源

### 署名格式

**在Markdown中**：
```markdown
![描述](path/to/image.jpg)

*图片：[标题]。[日期]。[创作者]。来源：[机构/档案馆]。[许可]。[网址]*
```

**示例**：
```markdown
![林则徐画像](images/photos/people/lin-zexu.jpg)

*图片：林则徐画像。约1840年。作者不详。来源：维基共享资源。公有领域。https://commons.wikimedia.org/wiki/File:Lin_Zexu.jpg*
```

## 创建原创视觉材料

### 地图

**工具**：
- QGIS（免费、开源）
- Adobe Illustrator
- Inkscape（免费）

**数据来源**：
- 历史地图集
- 学术出版物
- 政府调查

**最佳实践**：
- 使用准确的历史边界
- 清晰的图例和标签
- 一致的配色方案
- 包含比例尺和方向

### 信息图表

**工具**：
- Canva（有免费版）
- Adobe Illustrator
- Inkscape

**设计原则**：
- 清晰的视觉层次
- 准确的数据表示
- 可访问的颜色选择
- 数据的适当引用

### 时间线可视化

**工具**：
- TimelineJS（免费、基于网页）
- Adobe Illustrator
- 自定义HTML/CSS/JavaScript

**内容**：
- 带日期的主要事件
- 简要描述
- 链接到详细页面
- 可用时包含图片

## 法律和伦理考虑

### 敏感内容

**战争暴行**：
- 为历史准确性而包含
- 提供内容警告
- 谨慎使用图形图像
- 关注教育价值

- 尊重所有观点

**何时寻求许可**：
- 近期照片（1928年后）
- 版权状态不明
- 商业档案
- 私人收藏

**如何请求**：
- 介绍自己和项目
- 解释教育目的
- 说明预期用途
- 提供署名
- 准备支付费用（如需要）
