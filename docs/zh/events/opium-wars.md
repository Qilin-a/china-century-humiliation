# 鸦片战争 (1839-1842, 1856-1860)

鸦片战争是中国与西方列强（主要是英国）之间的两次冲突，标志着百年国耻的开始。这些战争从根本上改变了中国与外部世界的关系，暴露了清朝的软弱。

---

## 背景

### 鸦片贸易

**起源**：
- 英国东印度公司在印度种植鸦片
- 鸦片出口到中国以平衡贸易逆差
- 从1780年代起中国对鸦片的需求迅速增长
- 白银流出中国用于支付鸦片

**问题规模**：
- 1729年：清政府禁止吸食鸦片
- 1800年：皇帝颁布禁止进口鸦片的诏令
- 尽管有禁令，走私急剧增加
- 到1838年：估计每年进口40,000箱（2,400吨）
- 数百万中国人鸦片成瘾
- 严重的经济和社会后果

**中国的回应**：
- 多次试图压制贸易均告失败
- 官员腐败助长走私
- 1838年：道光皇帝任命林则徐为钦差大臣
- 林则徐的使命：在广州消除鸦片贸易

---

## 第一次鸦片战争 (1839-1842)

### 直接原因

**林则徐的行动（1839年）**：
- 1839年3月：林则徐抵达广州
- 要求外国商人交出所有鸦片
- 将外国商人限制在商馆内
- 威胁切断食物和水供应
- 1839年6月：销毁20,000多箱鸦片（1,400吨）
- 强迫英国商人签署保证书，承诺不再进行鸦片贸易

**英国的回应**：
- 英国商人向伦敦申请赔偿
- 英国政府要求赔偿
- 英国议会辩论（以271-262票决定开战）
- 1840年6月：英国远征军抵达中国

### 军事行动

**英国优势**：
- 现代蒸汽动力战舰
- 优越的海军火炮
- 专业的军事组织
- 更好的后勤和通讯

**中国劣势**：
- 过时的武器和战术
- 军事单位之间协调不良
- 指挥系统腐败和无能
- 沿海防御不足

**关键战役**：

1. **穿鼻之战（1841年1月）**
   - 英军摧毁中国战船
   - 占领穿鼻炮台
   - 展示英国海军优势

2. **广州之战（1841年5月）**
   - 英军占领广州周围炮台
   - 城市以600万银元赎回
   - 林则徐被革职流放

3. **长江战役（1842年）**
   - 英军北上，占领沿海城市
   - 占领宁波、上海、镇江
   - 威胁南京
   - 切断大运河，威胁北京粮食供应

### 《南京条约》（1842年8月29日）

**领土条款**：
- 香港岛"永久"割让给英国
- 在中国领土上建立英国殖民地

**商业条款**：
- 开放五个通商口岸：广州、厦门、福州、宁波、上海
- 固定关税为5%从价税
- 废除公行垄断制度
- 英国商人可与中国商人自由贸易

**财政条款**：
- 赔款2100万银元：
  - 600万用于被销毁的鸦片
  - 300万用于英国商人的债务
  - 1200万用于战争费用

**外交条款**：
- 国家间正式平等
- 政府间直接沟通
- 结束叩头礼仪要求

**《五口通商附粘善后条款》（1843年）**：
- 英国臣民享有治外法权
- 最惠国待遇条款
- 在通商口岸设立英国领事

### 后果

**对中国**：
- 第一个不平等条约
- 丧失主权和领土
- 为未来外国要求开创先例
- 暴露军事弱点
- 通商口岸经济混乱

**对英国**：
- 确保进入中国市场
- 香港成为主要贸易中心
- 为其他列强树立榜样
- 鸦片贸易继续并扩大

**国际影响**：
- 其他列强要求类似条约
- 1844年：《望厦条约》（美国）
- 1844年：《黄埔条约》（法国）
- 通商口岸制度的开始

## 第二次鸦片战争（英法联军之役，1856-1860）

### 原因

**直接导火索 - 亚罗号事件（1856年10月）**：
- 中国官员登上亚罗号，一艘中国人拥有的船
- 该船在香港注册，悬挂英国国旗
- 12名中国船员因海盗罪被捕
- 英国声称侵犯英国主权
- 要求道歉并释放船员
- 中国拒绝，声称该船注册已过期

**根本原因**：
- 英国对《南京条约》执行不满
- 渴望更大程度进入中国市场
- 法国寻求扩大影响力（以传教士被杀为借口）
- 俄国和美国的机会主义

**英国要求**：
- 在北京设立外交代表
- 修订商业条约
- 鸦片贸易合法化
- 向外国开放整个中国
- 中国拒绝所有要求

### 军事行动

**联军**：
- 英法结成联盟
- 俄美作为"中立"方（后来获益）
- 现代军事技术和战术
- 协调的海陆作战

**第一阶段（1856-1858）**：

1. **广州战役（1856-1857）**
   - 联军炮轰广州
   - 1857年12月占领城市
   - 两广总督叶名琛被俘

2. **大沽炮台（1858年5月）**
   - 联军舰队攻击天津炮台
   - 激战后占领炮台
   - 对北京的威胁迫使谈判

3. **《天津条约》（1858年6月）**
   - 在军事压力下签署
   - 中国同意广泛让步
   - 清廷希望避免批准

**第二阶段（1859-1860）**：

4. **大沽炮台之败（1859年6月）**
   - 英国舰队试图强行通过
   - 中国防御已加强
   - 英国遭受罕见失败
   - 464名英军伤亡

5. **联军升级（1860年）**
   - 派遣更大规模远征军
   - 18,000名英法军队
   - 1860年8月：占领大沽炮台
   - 1860年9月：占领天津

6. **进军北京（1860年9-10月）**
   - 联军向首都推进
   - 八里桥之战（9月21日）
   - 中国军队溃败
   - 咸丰皇帝逃往满洲

7. **洗劫圆明园（1860年10月）**
   - 联军劫掠圆明园（颐和园）
   - 珍宝被盗或毁坏
   - 10月18-19日：额尔金勋爵下令焚毁宫殿
   - 报复中国虐待俘虏
   - 象征性摧毁中国帝国威望

### 《天津条约》（1858）和《北京条约》（1860）

**《天津条约》（1858）**：

*政治*：
- 外国公使驻京
- 外国人可在中国内地旅行
- 保护基督教传教士
- 中国不得使用"夷"字称呼外国人

*商业*：
- 增开10个通商口岸
- 外国船只可在长江航行
- 降低内地通行税
- 鸦片贸易合法化（在1858年税则中）

*财政*：
- 向英国赔款400万两
- 向法国赔款200万两
- 赔偿英法商人

**《北京条约》（1860）**：

*与英国*：
- 割让九龙半岛
- 赔款增至800万两
- 天津开为通商口岸
- 中国劳工可移民

*与法国*：
- 赔款增至800万两
- 归还天主教会财产
- 法国传教士可在中国各地传教

*与俄国*：
- 割让乌苏里地区（滨海省份）
- 超过40万平方英里领土
- 俄国不战而获

### 后果

**直接影响**：
- 中国被迫接受外国外交存在
- 鸦片贸易合法化
- 基督教传教士受保护
- 向俄国大量领土损失
- 主权进一步侵蚀

**长期影响**：
- 加深中国屈辱
- 证明抵抗无用
- 鼓励其他列强寻求特权
- 削弱清朝合法性
- 促成太平天国运动继续
- 引发洋务运动

**文化影响**：
- 圆明园的毁灭成为屈辱的象征
- 文化珍宝损失（许多仍在西方博物馆）
- 中国精英的心理创伤
- 强化对西方野蛮的认知

## 历史意义

### 军事教训

**中国弱点暴露**：
- 技术落后
- 军事组织不良
- 指挥结构腐败
- 后勤不足
- 过时的战术和战略

**现代化必要性**：
- 认识到需要西方技术
- 洋务运动的开始
- 试图使军队现代化
- 由于系统性问题成功有限

### 外交革命

**朝贡体系终结**：
- 传统中国世界观受到挑战
- 被迫接受威斯特伐利亚国家体系
- 平等外交关系
- 融入国际法体系

**通商口岸制度**：
- 中国城市中的外国飞地
- 治外法权
- 丧失关税自主权
- 半殖民地地位

### 经济影响

**鸦片危机**：
- 持续大量鸦片进口
- 白银外流加速
- 成瘾造成社会破坏
- 经济混乱

**贸易失衡**：
- 外国商品涌入中国市场
- 传统产业受损
- 经济依赖外国列强
- 丧失经济主权

### 社会和政治影响

**清朝合法性危机**：
- 无力保护人民
- 无法抵抗外国侵略
- 失去天命
- 革命运动兴起

**思想回应**：
- 质疑传统价值观
- 对西学感兴趣
- 关于现代化的辩论
- 改革运动的开始

## 史学争论

### 英国视角（历史）
- 对中国阻碍贸易的合法回应
- 执行条约权利
- 将中国带入"文明"国际社会
- 大英帝国的经济必要性

### 中国视角
- 赤裸裸的侵略和帝国主义
- 外国列强贩毒
- 侵犯中国主权
- 民族屈辱的开始

### 现代学术观点
- 多种因素的复杂互动
- 双方都有责任
- 不兼容体系的冲突
- 误解和冲突的悲剧
- 鸦片贸易在道德上无法辩护
- 中国的抵抗可以理解但徒劳

---

## 遗产

鸦片战争在中国历史记忆中仍然具有深远意义：

1. **屈辱的象征**：代表中国衰落的开始
2. **反帝国主义**：现代中国民族主义的基础
3. **现代化辩论**：引发关于传统与现代的持续讨论
4. **外交关系**：影响中国对主权和国际法的态度
5. **历史教育**：中国爱国主义教育的核心
6. **文化记忆**：圆明园遗址作为提醒被保存

这些战争展示了在现代世界中技术和组织落后的后果，使中国走上了一条痛苦的适应和革命道路，这条道路持续了一个多世纪。
