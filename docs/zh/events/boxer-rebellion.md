# 义和团运动与八国联军侵华 (1899-1901)

义和团运动是中国的一场反洋、反教运动，导致八个外国列强的干预。随后的《辛丑条约》对中国施加了迄今为止最严厉的条款，代表了百年国耻的最低点之一。

---

## 背景

### 义和团运动的起源

**义和拳会（义和拳）**：

- 俗称："拳民"或"义和团"（因练习武术）
- 起源于山东省（1890年代末）
- 结合武术、附体仪式和民间宗教
- 相信自己拥有神力，刀枪不入
- 口号："扶清灭洋"

**社会和经济原因**：

- 自然灾害：华北地区的旱灾和水灾（1898-1900年）
- 经济困难和饥荒
- 外国竞争导致失业
- 对外国经济特权的怨恨
- 传统社会的破坏

**反洋情绪**：

- 对甲午战争后领土让步的愤怒
- 对基督教传教士和信徒的怨恨
- 外国控制铁路和矿山
- 治外法权和不平等条约
- 对外国文化帝国主义的认知

### 运动的发展

**初期阶段（1898-1899年）**：

- 最初是反清秘密会社
- 逐渐转向反洋焦点
- 从山东蔓延到直隶（河北）省
- 攻击中国基督徒和传教士
- 破坏外国财产（铁路、电报线）

**清政府的回应**：

- 最初试图镇压义和团
- 保守派官员同情反洋信息
- 慈禧太后看到利用义和团对抗外国人的机会
- 1900年1月：慈禧发布诏书使义和团合法化
- 政府从镇压转向支持

---

## 危机升级（1900年春夏）

### 义和团活动

**1900年5-6月**：

- 义和团大量进入北京和天津
- 对中国基督徒的攻击加剧
- 外国财产被毁
- 铁路和电报线被切断
- 北京外国使馆受到威胁

**对基督徒的暴力**：

- 数千名中国基督徒被杀
- 外国传教士被谋杀
- 教堂被烧毁
- 在山西巡抚毓贤治下特别严重
- 估计全国有30,000-50,000名中国基督徒被杀

### 外国回应

**军事集结**：

- 外国列强向大沽炮台派遣海军力量
- 派遣海军陆战队保护北京使馆
- 5月31日：400多名外国军队抵达北京
- 6月10日：西摩尔海军上将的救援远征队从天津出发（2,000人）
- 西摩尔部队被义和团阻挡，不得不撤退

**使馆区围攻**：

- 1900年6月20日：德国公使克林德男爵在北京被暗杀
- 同日：清廷向外国列强宣战
- 6月20日-8月14日：北京使馆区围攻
- 约900名外国平民和士兵被围困
- 还有3,000多名中国基督徒在使馆避难
- 防御周长约3/4英里

## 八国联军干预

### 联军组成

**参与国家**：

1. 日本（最大派遣：约20,000人）
2. 俄国（约13,000人）
3. 英国（约12,000人）
4. 法国（约3,000人）
5. 美国（约3,000人）
6. 德国（最初约900人，后来更多）
7. 奥匈帝国（约300人）
8. 意大利（约80人）

**总兵力**：约55,000人

### 军事行动

**攻占大沽炮台（1900年6月17日）**：

- 联军海军攻占炮台
- 为军队登陆开辟道路
- 清廷将此视为战争行为

**天津之战（1900年7月13-14日）**：

- 联军经过激战攻占天津
- 城市被劫掠和焚烧
- 建立进军北京的基地

**进军北京（1900年8月）**：

- 约20,000人的联军从天津出发（8月4日）
- 与清军和义和团作战多次
- 北仓之战（8月5日）
- 由于清军组织不良，抵抗相对较轻

**解救北京（1900年8月14日）**：

- 联军进入北京
- 55天围攻后使馆区解围
- 慈禧太后和光绪皇帝逃往西安
- 联军占领北京

### 暴行和劫掠

**联军行为**：

- 对北京的广泛劫掠
- 紫禁城被掠夺
- 文物被毁
- 杀害中国平民
- 对民众的强奸和暴力
- 对周边地区的惩罚性远征

**德国"匈奴演说"**：

- 德皇威廉二世对德军的臭名昭著演说
- 敦促军队像"阿提拉统治下的匈奴人"一样无情
- "不给予宽恕，不接受俘虏"
- 导致战役的残暴行为

**破坏规模**：

- 估计中国死亡人数：50,000-100,000+
- 许多村庄在惩罚性远征中被毁
- 文化珍宝被盗或毁坏
- 受影响地区经济破坏

## 《辛丑条约》（1901年9月7日）

### 谈判

**中国谈判代表**：

- 李鸿章（在谈判期间去世，1900年11月）
- 庆亲王（奕劻）

**外国列强**：八国代表加上西班牙、比利时、荷兰

### 主要条款

**1. 赔款**：

- **4.5亿两白银**（约18,000吨）
- 相当于约3.33亿美元
- 39年分期偿还，年息4%
- **本息合计：9.82亿两**
- 中国历史上最大赔款
- 以盐税和海关收入作担保
- 人均：约每个中国人1两

**2. 惩处官员**：

- 处决10名支持义和团的高级官员
- 120多名官员被处死、流放或革职
- 山西巡抚毓贤因杀害传教士被处决
- 对官员的追封降级
- 45个城市禁止科举考试5年

**3. 军事条款**：

- 拆毁大沽炮台和其他防御工事
- 外国军队驻扎北京（使馆卫队）
- 外国军队沿北京至海铁路驻扎
- 禁止进口军火2年
- 外国军队永久驻扎中国首都

**4. 使馆区**：

- 北京使馆区由外国独占控制
- 中国人不得在该区居住
- 允许设防
- 实际上是中国首都内的外国领土

**5. 外交**：

- 向德国和日本正式道歉和派遣谢罪使团
- 为被杀外国人建立纪念碑
- 在外国人被杀地区暂停官方考试

**6. 反洋组织**：

- 参加反洋会社者处死刑
- 官员对其管辖区内的反洋暴力负责
- 严格禁止反洋活动

### 赔款分配

**各国份额**：

- 俄国：28.97%
- 德国：20.02%
- 法国：15.75%
- 英国：11.25%
- 日本：7.73%
- 美国：7.32%
- 意大利：5.91%
- 比利时：1.89%
- 奥匈帝国：0.89%
- 其他：0.27%

## 后果

### 对中国

**财政灾难**：

- 赔款相当于政府2-3年收入
- 需要大幅增税
- 被迫依赖外国贷款
- 关税和盐税收入抵押给外国列强
- 数十年财政瘫痪

**主权丧失**：

- 外国军队永久驻扎首都
- 使馆区在外国控制下
- 外国控制海关收入
- 无法进口武器
- 实际上的半殖民地地位

**政治影响**：

- 证明清朝完全无力抵抗外国列强
- 保守派失信
- 导致迟来的改革努力（新政，1901-1911年）
- 促进革命情绪
- 加速清朝灭亡（1911年）

**社会影响**：

- 深刻的民族屈辱
- 加强反满情绪
- 增加对激进改革或革命的支持
- 集体记忆中的创伤

### 国际后果

**美国"门户开放"政策**：

- 国务卿海约翰的通告（1899-1900年）
- 倡导所有列强平等商业准入
- 反对瓜分中国
- 确立美国对华政策框架

**赔款退还**：

- **美国**（1908年）：退还部分用于中国学生教育
  - 建立清华大学
  - 庚子赔款奖学金计划
- **英国**（1922年）：退还部分用于教育目的
- **其他国家**：各种部分退还或减免

**权力动态**：

- 日本成为在华主要强国
- 俄日在满洲竞争影响力
- 导致日俄战争（1904-1905年）
- 日本在东亚主导地位的开始

## 历史争论

### 义和团运动的性质

**爱国抵抗观点**：

- 反帝国主义的民众运动
- 对外国侵略的合法抵抗
- 被压迫人民的自发起义
- 捍卫中国主权的英勇行为

**排外暴力观点**：

- 非理性、迷信的运动
- 对外国人和基督徒的无差别暴力
- 被保守派清朝官员操纵
- 中国现代化的倒退

**平衡视角**：

- 具有多种原因的复杂运动
- 对帝国主义的合理不满
- 方法包括过度暴力
- 既有爱国也有排外因素
- 对中国的悲剧性后果

### 清政府责任

**罪责论据**：

- 玩世不恭地利用义和团为己所用
- 向八国宣战是鲁莽的
- 未能保护外国外交官
- 对暴力负有共同责任

**减轻因素**：

- 失去对局势的控制
- 宫廷在保守派和温和派之间分裂
- 面临没有好选择的不可能局面
- 一些官员保护外国人

### 外国行为

**合法防御**：

- 有权保护外交官和国民
- 对攻击和围攻作出回应
- 解救被围困的使馆

**过度报复**：

- 惩罚性远征杀害许多平民
- 广泛劫掠和破坏
- 过度的赔款要求
- 对中国人民的集体惩罚

## 遗产

### 历史记忆

**在中国**：

- 外国帝国主义和侵略的象征
- 民族屈辱的例子
- 对迷信和落后的警告
- 关于爱国主义与排外主义的辩论
- 爱国主义教育课程的一部分

**在西方**：

- 通常被描绘为"解救"被围困的外交官
- 较少强调惩罚性方面
- 这一时期的文物在西方博物馆
- 关于中国的刻板印象来源

### 长期影响

**对中国政治**：

- 促成清朝灭亡
- 加强革命运动
- 影响中国民族主义
- 塑造对外国干预的态度

**对国际关系**：

- 证明中国的软弱
- 鼓励进一步外国侵占
- 为集体干预开创先例
- 影响国际法发展

**文化影响**：

- 文化珍宝损失
- 民族意识中的创伤
- 影响中国对现代化的看法
- 塑造关于传统与现代的辩论

## 与其他事件的比较

**与鸦片战争的相似之处**：

- 外国军事干预
- 强加不平等条约
- 主权丧失
- 民族屈辱

**独特方面**：

- 民众反洋运动（不仅是政府）
- 八国联盟（前所未有的规模）
- 中国历史上最大赔款
- 外国军队驻扎首都
- 百年国耻最严厉条款

## 教训与反思

### 对中国

1. 排外和迷信无法击败现代列强
2. 需要真正的现代化，而非仅仅反洋
3. 理性外交政策的重要性
4. 操纵民众运动的危险

### 对国际社会

1. 集体惩罚有长期后果
2. 过度要求造成持久怨恨
3. 国际关系中的文化敏感性
4. 保护国民与尊重主权之间的平衡

### 普遍教训

1. 极端民族主义和排外主义的危险
2. 帝国主义和外国统治的后果
3. 解决合理不满的重要性
4. 使用军事力量时需要克制
