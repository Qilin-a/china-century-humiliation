# 五四运动 (1919)

五四运动是一场文化和政治运动，起因是中国在第一次世界大战后巴黎和会上受到的不公正待遇。它标志着中国现代史的转折点，将反帝国主义的民族主义与拒绝传统儒家价值观的激进文化革命结合在一起。

---

## 背景

### 第一次世界大战与中国

**中国的参与**：
- 1917年8月：中国向德国和奥匈帝国宣战
- 派遣14万多名劳工到欧洲（华工）
- 支持协约国战争努力
- 期待在和会上得到公平待遇

**中国的期望**：
- 收回德国在山东省的租借地
- 废除不平等条约
- 被承认为平等国家
- 结束外国在华特权

### 巴黎和会（1919年）

**中国代表团**：
- 由外交总长陆征祥率领
- 提出"七项要求"：
  1. 废除势力范围
  2. 撤退外国军队和警察
  3. 废除治外法权
  4. 归还外国租界
  5. 归还租借地
  6. 废除二十一条（日本，1915年）
  7. 归还关税自主权

**山东问题**：
- 德国自1898年起控制胶州湾和青岛
- 日本于1914年夺取这些领土
- 秘密条约（1917-1918年）：协约国承诺将山东交给日本
- 中国要求直接归还德国租借地
- 日本坚持保留山东

**凡尔赛条约决定**：
- **1919年4月30日**：决定公布
- 山东转让给日本，而非归还中国
- 中国要求被拒绝
- 中国虽在战胜国一方却被当作战败国对待
- 中国国内深感震惊和屈辱

---

## 五四事件（1919年5月4日）

### 北京学生抗议

**直接导火索**：
- 凡尔赛和会决定的消息传到北京
- 学生组织大规模抗议
- 对外国列强和中国政府的愤怒

**1919年5月4日示威**：
- **3,000多名学生**来自北京13所大学在天安门集会
- 北京大学学生领导抗议
- 口号：
  - "拒绝在凡尔赛条约上签字！"
  - "废除二十一条！"
  - "抵制日货！"
  - "惩办国贼！"
  - "外争主权，内除国贼！"

**暴力和逮捕**：
- 学生游行至使馆区
- 攻击曹汝霖（亲日官员）住宅
- 火烧其宅邸
- 殴打章宗祥（驻日公使）
- 警察逮捕32名学生
- 政府开始镇压

### 运动的蔓延

**全国性抗议（1919年5-6月）**：
- 蔓延至200多个城市
- 上海、天津、武汉、广州学生加入
- 工人罢工
- 商人罢市（三罢：罢课、罢工、罢市）
- 成为全国性运动

**参与者**：
- **学生**：主要组织者和活动家
- **工人**：上海工人大罢工（6月5日）
- **商人**：商业罢市和抵制
- **知识分子**：作家、教授、记者
- **妇女**：积极参与，要求性别平等

**政府回应**：
- 最初镇压：逮捕、审查
- 6月3-4日：大规模逮捕学生（超过1,000人）
- 公众压力迫使政府让步
- 6月10日：释放被捕学生
- 6月28日：中国拒绝签署凡尔赛条约

## 新文化运动

### 思想革命

**起源（1915-1919年）**：
- 先于五四政治抗议
- 由陈独秀、胡适、鲁迅等知识分子领导
- 以《新青年》杂志为平台
- 批判中国传统文化

**核心思想**：
- **赛先生**（科学）：科学方法和理性主义
- **德先生**（民主）：民主治理和个人权利
- 拒绝儒家思想为封建和落后
- 拥抱西方思想和价值观
- 文学革命：白话文对文言文

**关键人物**：

1. **陈独秀**：
   - 创办《新青年》杂志（1915年）
   - 北京大学文科学长
   - 后来成为中国共产党联合创始人（1921年）

2. **胡适**：
   - 倡导文学革命
   - 推广白话文
   - 实用主义哲学家

3. **鲁迅**：
   - 最伟大的现代中国作家
   - 《狂人日记》（1918年）：第一篇白话文小说
   - 对传统社会的尖锐批判

4. **李大钊**：
   - 北京大学图书馆主任
   - 将马克思主义引入中国
   - 中共联合创始人

### 文化转型

**文学革命**：
- 白话文取代文言文成为文学语言
- 新形式：现代诗歌、短篇小说、长篇小说
- 可及性：普通人能读能写
- 报纸和杂志使用白话文

**倡导的社会改革**：
- 妇女权利和性别平等
- 结束包办婚姻
- 废除缠足
- 教育改革
- 个人自由
- 质疑家庭等级制度

**意识形态多样性**：
- 自由主义和民主
- 无政府主义
- 社会主义和马克思主义
- 实用主义
- 激烈的思想辩论

## 政治后果

### 直接结果

**外交胜利**：
- 中国拒绝签署凡尔赛条约（1919年6月28日）
- 三名亲日官员被免职：
  - 曹汝霖（交通总长）
  - 陆宗舆（币制局总裁）
  - 章宗祥（驻日公使）
- 抗议者的象征性胜利

**局限性**：
- 山东仍归日本
- 不平等条约依然存在
- 外国特权继续
- 军阀政府仍然软弱

### 长期政治影响

**政治运动的兴起**：

1. **中国共产党（CCP）**：
   - 1921年7月成立
   - 许多五四活动家成为共产党员
   - 陈独秀：首任中共总书记
   - 李大钊：联合创始人
   - 毛泽东：参与五四活动

2. **改组后的国民党（KMT）**：
   - 孙中山改组政党（1919-1924年）
   - 采纳更激进的纲领
   - 与中共第一次国共合作（1924-1927年）

**政治思想转变**：
- 对西方民主的幻灭
- 对苏联模式的兴趣
- 强调群众动员
- 反帝国主义成为核心
- 引入阶级斗争概念

### 民族主义

**新形式的民族主义**：
- 大众化、群众性（不仅是精英）
- 反帝国主义焦点
- 文化自信与现代化并存
- 学生行动主义成为政治力量
- 抵制作为政治武器

**反日情绪**：
- 因山东问题而加剧
- 抵制日货
- 后来抵抗运动的基础
- 塑造中日关系

### 现代中国的转折点

**政治**：
- 标志着大众政治的开始
- 学生成为政治力量
- 后来运动的基础
- 促成中共成立

**文化**：
- 与传统文化决裂
- 语言和文学现代化
- 思想解放
- 质疑权威

**社会**：
- 妇女解放运动
- 个人权利意识
- 社会流动
- 城乡分化

### 与其他运动的比较

**类似于**：
- 欧洲启蒙运动（理性主义、科学）
- 法国大革命（激进文化变革）
- 俄国革命（政治激进主义）

**独特方面**：
- 反帝国主义与文化革命结合
- 学生领导的群众运动
- 白话文革命
- 民族主义与国际主义的综合

## 争论和解释

### 在中国大陆（中华人民共和国）

**官方叙事**：
- 爱国、反帝国主义运动
- 新民主主义革命的开端
- 为中共成立准备条件
- 世界无产阶级革命的一部分
- 5月4日被定为"青年节"

**强调**：
- 反帝反封建
- 群众动员
- 革命潜力
- 与共产主义运动的联系

### 在台湾（中华民国）

**替代观点**：
- 文化复兴
- 民主和自由理想
- 与孙中山原则的联系
- 现代化运动
- 较少强调阶级斗争

### 西方学术界

**学术观点**：
- 复杂、多面的运动
- 既有进步也有问题的方面
- 拒绝传统有代价
- 促成政治激进化
- 文化破坏过度

**争论**：
- 拒绝传统是否必要？
- 是否为极权主义铺平道路？
- 传统与现代的平衡
- 知识分子在政治中的角色

## 结论

五四运动是中国现代史上的分水岭，将反帝国主义的民族主义与激进的文化转型结合在一起。它源于中国在凡尔赛和会上的屈辱，但发展成为对传统社会和文化的全面挑战。

运动的双重性质——政治抗议和文化革命——使其具有独特的影响力。它以前所未有的方式动员了学生、工人和商人，建立了新的政治行动主义形式，从根本上改变了中国的语言和文学。

虽然运动取得的直接政治成功有限，但其长期影响深远。它促成了中国共产党的成立，激发了中国民族主义，使中国文化现代化，并建立了延续至今的学生行动主义模式。

五四运动既代表了中国对百年国耻回应的高潮，也代表了最终将改变中国的新革命运动的开始。其遗产仍有争议，但无疑是理解现代中国的核心。
