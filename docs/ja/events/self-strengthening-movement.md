# 洋務運動 (1861-1895)

洋務運動（ようむうんどう）は、アヘン戦争と太平天国の乱での軍事的敗北を受けて、清朝末期に開始された一連の制度改革でした。この運動は、伝統的な儒教的価値観を保持しながら西洋の方法を採用することで、中国の軍事、経済、技術の近代化を目指し、「中体西用」（中国の学問を根本とし、西洋の学問を実用とする）というスローガンに集約されました。

**不平等条約**：

- 外国租界と治外法権
- 賠償金支払い
- 貿易の強制的開放

**内政上の課題**：
- 太平天国の乱による荒廃
- その他の反乱（捻軍、回民蜂起）
- 財政危機
- 行政の腐敗
- 社会不安

### 思想的基盤

**西洋の実力の認識**：
- 西洋の軍事的優位の承認
- 工業力の理解
- 技術格差の認識
- 選択的近代化の必要性

**指導理念**：
- **「中体西用」**
- 儒教的道徳と社会秩序の保持
- 西洋技術と方法の採用
- 技術移転であって文化変革ではない
- 選択的近代化

## 主要プログラムと機関

### 軍事近代化

**軍工廠と造船所**：

1. **江南製造局**（上海、1865年）
   - 中国最大の軍工廠
   - 近代的武器と船舶を生産
   - 西洋書籍翻訳局
   - 外国専門家を雇用

2. **福州船政局**（1866年）
   - 近代的軍艦を建造
   - 海軍学校を併設
   - フランスの技術援助
   - 海軍将校を訓練

3. **天津機器局**（1867年）
   - 弾薬と武器を生産
   - 化学品生産
   - 近代的製造技術

**近代海軍**：
- **北洋艦隊**（北洋水師）
  - 李鴻章のプロジェクト
  - ヨーロッパから近代的軍艦を購入
  - 1888年までに世界最大級の海軍の一つ

- **南洋艦隊**（南洋水師）
- **福建艦隊**

**陸軍改革**：
- 西洋式訓練
- 近代的武器
- 新しい軍事学校
- 外国軍事顧問

### 工業発展

1. **開平炭鉱**（1877年）

   - 工業への燃料供給

2. **漢陽鉄廠**（1890年）

**軽工業**：
- 紡績工場
- マッチ工場
- 製紙工場
- 近代的印刷

## 指導原則

### 「中体西用」

**意味**：
- 儒教倫理と社会秩序の保持
- 西洋技術と方法の採用
- 文化変革なしの技術移転
- 選択的近代化

**前提**：
- 技術は文化から分離可能
- 西洋の強さは技術のみにある
- 中国文化は道徳的に優越
- 根本的変革なしに近代化可能

**限界**：
- 制度的要因の過小評価
- 政治・社会改革の無視
- 技術には支援制度が必要
- 変革への文化的抵抗

### 「官督商弁」

**ビジネスモデル**：
- 政府が資本と監督を提供
- 商人が日常運営を管理
- 官の支援と商の効率を結合
- 外国競争からの保護

**問題**：
- 官僚の干渉と腐敗
- 権限の不明確さ
- 非効率性
- 民間主導性の制限
- 官商間の対立

- 法制改革なし
- 伝統的科挙制度は不変
- 儒教イデオロギーは挑戦されず

**財政的制約**：
- 限られた政府収入
- 競合する需要（軍事、賠償金）
- 腐敗と浪費
- 投資不足
- 外国借款への依存

**行政上の問題**：
- 分散的実施
- 地域的変動
- 調整の欠如

## 日清戦争での敗北（1894-1895年）

### 近代化の試練

**期待**：
- 北洋艦隊は世界最大級の一つ
- 近代的武器と訓練
- 数十年の軍事近代化
- 中国の実力への自信

**現実**：
- 日本に惨敗
- 北洋艦隊壊滅
- 近代的武器が無効
- 指導力と戦術の貧弱
- 腐敗が軍を弱体化

**敗北の理由**：
- 日本のより包括的な改革
- 明治維新が制度を変革
- 中国の改革は表面的
- 腐敗（資金が頤和園建設に流用）
- 訓練と士気の低さ
- 時代遅れの戦術
- 政治的意志の欠如

### 敗北の影響

**弱点の露呈**：
- 洋務運動の不十分さ
- 技術だけでは不十分
- より深い改革の必要性
- 制度問題が重要

**政治的結果**：
- 下関条約（1895年）
- 台湾と朝鮮の喪失
- 巨額賠償金
- 威信の失墜
- 列強による分割競争

**思想的影響**：
- 洋務運動アプローチの信用失墜
- 政治改革への要求
- 改革運動の台頭
- 戊戌変法（1898年）
- 革命的感情

## 歴史的意義

### 成果

**近代化の基礎**：
- 最初の体系的近代化努力
- 工業基盤の創設
- 技術者の養成
- 西洋知識の導入
- 改革の先例

**部分的成功**：
- いくつかの産業を確立
- 軍事の部分的近代化
- インフラの開始
- 教育改革の着手
- 外交の近代化

### 根本的限界

**概念的欠陥**：
- 「中体西用」では不十分
- 技術は制度と分離不可能
- 必要な変革の範囲を過小評価
- 文化保守主義が制限

**構造的問題**：
- 政治改革なし
- 社会変革なし
- エリート主導、大衆基盤なし
- 地域的、全国的でない
- 表面的、根本的でない

### 明治日本との比較

**類似点**：
- 両国とも西洋の圧力に直面
- 両国とも近代化を求める
- 両国とも西洋技術を採用
- 両国とも独立維持を目指す

**相違点**：

| 側面 | 中国 | 日本 |
|------|------|------|
| 政治改革 | なし | 根本的（明治維新） |
| 社会改革 | 最小限 | 広範囲（身分制度廃止） |
| 教育 | 限定的、エリート | 普遍的、大衆 |
| 範囲 | 地域的 | 全国的 |
| 指導力 | 分裂 | 統一 |
| イデオロギー | 伝統保持 | 社会変革 |
| 結果 | 敗北（1895年） | 勝利（1895年） |

**教訓**：
- 包括的改革が必要
- 政治・社会変革が不可欠
- 技術には制度的支援が必要
- 中途半端では不十分
- 国家統一が重要

## 結論

洋務運動は、西洋の挑戦に対する中国初の体系的近代化の試みでした。軍事・工業発展でいくつかの成功を収め、西洋知識を導入し、将来の改革の基礎を築きました。しかし、その根本的限界—技術は制度的・文化的変革なしに採用できるという信念—が最終的な失敗を運命づけました。

日清戦争（1894-1895年）での惨敗は、洋務運動アプローチの不適切さを露呈しました。政治的、社会的、経済的制度を変革した日本の包括的な明治維新が、中国の限定的な技術借用よりもはるかに効果的であることを実証しました。

この運動の失敗は重要な教訓を与えました：近代化には包括的改革が必要、技術は制度から分離できない、中途半端では不十分。これらの教訓は後続の改革努力に影響を与え、最終的に20世紀の中国の革命的変革に貢献しました。

その限界にもかかわらず、洋務運動は中国の近代的変革における重要な段階であり、中国が近代世界で生き残るために西洋から学ぶ必要があることを最初に認識したものでした。
