# 新文化運動 (1915-1920年代)

新文化運動（しんぶんかうんどう）は、伝統的な儒教的価値観に挑戦し、近代化、科学、民主主義、個人主義を推進した中国の革命的知的・文化運動でした。1915年頃に始まり、1919年の五・四運動で頂点に達したこの運動は、中国の知的生活を根本的に変革し、20世紀中国を再形成する政治的・社会的革命の基礎を築きました。

---

## 背景と起源

### 失敗の文脈

**失敗した政治改革**：
- 戊戌変法の粉砕（1898年）
- 義和団事件の余波（1900-1901年）
- 1911年革命が共和国を樹立したが社会変革に失敗
- 袁世凱の独裁と帝政復活の試み（1915-1916年）
- 軍閥時代の混乱

**知的危機**：
- 政治変革だけでは不十分という認識
- 伝統文化が中国の弱さの根源と見なされる
- 根本的な文化変革の必要性
- 西洋啓蒙思想の影響
- 急進的解決策を求める若い知識人

**国際的背景**：
- 第一次世界大戦（1914-1918年）
- ロシア革命（1917年）
- ウィルソン理想主義と民族自決
- 民主主義と社会主義思想の世界的拡散
- 西洋帝国主義への幻滅

## 結論

新文化運動は、中国史における最も深遠な知的・文化的変革の一つでした。伝統的儒教文明に根本的に挑戦し、科学、民主主義、個人解放に基づく新しい文化を創造しようとしました。すべての目標を達成せず、伝統の拒否において極端に走りましたが、中国の知的生活と文化を永続的に変革しました。

運動による白話文の推進は、文学と教育を民主化し、より広範な人々にアクセス可能にしました。伝統的倫理、特に女性と若者に関する批判は、社会変革への空間を開きました。西洋思想と批判的思考の導入は、様々な政治運動を主導する現代中国知識人を創出しました。

最も重要なことに、新文化運動は中国の20世紀革命の基盤を準備しました。知識人と学生の世代を急進化させ、その多くが中国共産党の創設者となりました。大衆政治運動のイデオロギー的・文化的条件を創出しました。新文化から成長した1919年の五・四運動は、現代中国革命政治の始まりを示しました。

新文化運動の遺産は論争的で複雑なままです。それは文化的啓蒙と偶像破壊的過剰、個人解放と社会分裂、近代化とアイデンティティ危機の両方を表しています。この運動を理解することは、現代中国知的史と中国の革命的変革の文化的基盤を理解するために不可欠です。
