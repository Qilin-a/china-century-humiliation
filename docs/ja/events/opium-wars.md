# アヘン戦争 (1839-1842, 1856-1860)

アヘン戦争は、中国と西洋列強（主にイギリス）との間で起こった二つの紛争で、屈辱の世紀の始まりを示すものでした。これらの戦争は、中国と外部世界との関係を根本的に変え、清朝の弱さを露呈しました。

---

## 背景

### アヘン貿易

**起源**：
- イギリス東インド会社がインドでアヘンを栽培
- 貿易赤字を埋めるため中国にアヘンを輸出
- 1780年代から中国のアヘン需要が急速に増加
- アヘン代金として銀が中国から流出

**問題の規模**：
- 1729年：清政府がアヘン喫煙を禁止
- 1800年：皇帝がアヘン輸入を禁止する勅令を発布
- 禁令にもかかわらず密輸が劇的に増加
- 1838年まで：推定年間40,000箱（2,400トン）が輸入
- 数百万人の中国人がアヘン中毒
- 深刻な経済的・社会的影響

**中国の対応**：
- 貿易を抑制する複数の試みが失敗
- 官僚の腐敗が密輸を助長
- 1838年：道光帝が林則徐を欽差大臣に任命
- 林の使命：広州でアヘン貿易を撲滅

---

## 第一次アヘン戦争 (1839-1842)

### 直接的原因

**林則徐の行動（1839年）**：
- 1839年3月：林が広州に到着
- 外国商人に全てのアヘンの引き渡しを要求
- 外国商人を商館に監禁
- 食料と水の供給を断つと脅迫
- 1839年6月：20,000箱以上のアヘン（1,400トン）を破壊
- イギリス商人にアヘン貿易をしないという誓約書への署名を強制

**イギリスの対応**：
- イギリス商人がロンドンに補償を要求
- イギリス政府が賠償を要求
- イギリス議会での討論（271対262で開戦を決定）
- 1840年6月：イギリス遠征軍が中国に到着

### 軍事作戦

**イギリスの優位性**：
- 近代的な蒸気動力軍艦
- 優れた海軍砲
- 専門的な軍事組織
- より良い兵站と通信

**中国の劣勢**：
- 時代遅れの武器と戦術
- 軍事部隊間の調整不足
- 指揮系統の腐敗と無能
- 不十分な沿岸防衛

**主要な戦闘**：

1. **穿鼻の戦い（1841年1月）**
   - イギリス軍が中国のジャンク船を破壊
   - 穿鼻砲台を占領
   - イギリス海軍の優位性を実証

2. **広州の戦い（1841年5月）**
   - イギリス軍が広州周辺の砲台を占領
   - 都市が600万銀ドルで身代金を支払う
   - 林則徐が罷免され追放される

3. **揚子江作戦（1842年）**
   - イギリス軍が北上し、沿岸都市を占領
   - 寧波、上海、鎮江を占領
   - 南京を脅かす
   - 大運河を遮断し、北京の食料供給を脅かす

### 南京条約（1842年8月29日）

**領土条項**：
- 香港島を「永久に」イギリスに割譲
- 中国領土にイギリス植民地を設立

**商業条項**：
- 5つの条約港を開放：広州、厦門、福州、寧波、上海
- 5%従価税の固定関税
- 公行独占制度の廃止
- イギリス商人が中国商人と自由に貿易できる

**財政条項**：
- 2100万銀ドルの賠償金：
  - 破壊されたアヘンに600万
  - イギリス商人への債務に300万
  - 戦争費用に1200万

**外交条項**：
- 国家間の公式な平等
- 政府間の直接的なコミュニケーション
- 叩頭の礼の要求の終了

**虎門寨追加条約（1843年）**：
- イギリス臣民の治外法権
- 最恵国条項
- 条約港にイギリス領事

### 結果

**中国にとって**：
- 最初の不平等条約
- 主権と領土の喪失
- 将来の外国要求の先例
- 軍事的弱さの露呈
- 条約港での経済混乱

**イギリスにとって**：
- 中国市場への商業的アクセスを確保
- 香港が主要貿易拠点となる
- 他の列強が従うモデル
- アヘン貿易が継続・拡大

**国際的影響**：
- 他の列強が同様の条約を要求
- 1844年：望厦条約（アメリカ）
- 1844年：黄埔条約（フランス）
- 条約港制度の始まり

## 第二次アヘン戦争（アロー戦争、1856-1860）

### 原因

**直接的きっかけ - アロー号事件（1856年10月）**：
- 中国当局がアロー号に乗船、中国人所有の船
- 香港で登録され、イギリス国旗を掲げていた
- 海賊行為で12人の中国人乗組員を逮捕
- イギリスがイギリス主権の侵害を主張
- 謝罪と乗組員の返還を要求
- 中国が拒否、船の登録が期限切れと主張

**根本的原因**：
- 南京条約の実施に対するイギリスの不満
- 中国市場へのより大きなアクセスへの欲求
- フランスが影響力拡大を求める（宣教師殺害を口実）
- ロシアとアメリカの機会主義

**イギリスの要求**：
- 北京への直接外交代表
- 商業条約の改訂
- アヘン貿易の合法化
- 全中国の外国貿易への開放
- 中国が全ての要求を拒否

### 軍事作戦

**連合軍**：
- イギリスとフランスが同盟を結成
- ロシアとアメリカが「中立」当事者として（後に利益を得る）
- 近代的な軍事技術と戦術
- 調整された海陸作戦

**第一段階（1856-1858）**：

1. **広州作戦（1856-1857）**
   - 連合軍が広州を砲撃
   - 1857年12月に都市を占領
   - 総督葉名琛を捕虜

2. **大沽砲台（1858年5月）**
   - 連合艦隊が天津の砲台を攻撃
   - 激戦の後に砲台を占領
   - 北京への脅威が交渉を強制

3. **天津条約（1858年6月）**
   - 軍事的圧力下で署名
   - 中国が広範な譲歩に同意
   - 清朝は批准を避けることを望む

**第二段階（1859-1860）**：

4. **大沽砲台の撃退（1859年6月）**
   - イギリス艦隊が強行通過を試みる
   - 中国の防衛が強化されていた
   - イギリスが稀な敗北を喫する
   - イギリス軍464人の死傷者

5. **連合軍のエスカレーション（1860年）**
   - より大規模な遠征軍を派遣
   - 18,000人のイギリス・フランス軍
   - 1860年8月：大沽砲台を占領
   - 1860年9月：天津を占領

6. **北京への進軍（1860年9-10月）**
   - 連合軍が首都に進撃
   - 八里橋の戦い（9月21日）
   - 中国軍が壊走
   - 咸豊帝が満州に逃亡

7. **円明園の略奪（1860年10月）**
   - 連合軍が円明園（旧夏宮殿）を略奪
   - 宝物が盗まれるか破壊される
   - 10月18-19日：エルギン卿の命令で宮殿を焼失
   - 中国による捕虜虐待への報復
   - 中国帝国の威信の象徴的破壊

### 天津条約（1858）と北京条約（1860）

**天津条約（1858）**：

*政治*：
- 北京に外国公使館
- 中国内陸部での外国人旅行
- キリスト教宣教師の保護
- 中国が外国人に「夷」という用語を使用しない

*商業*：
- さらに10の条約港を開放
- 揚子江での外国船
- 内陸通行税の削減
- アヘン貿易の合法化（1858年関税で）

*財政*：
- イギリスに400万両
- フランスに200万両
- イギリス・フランス商人への補償

**北京条約（1860）**：

*イギリスとの条約*：
- 九龍半島を割譲
- 賠償金を800万両に増額
- 天津を条約港として開放
- 中国人労働者の移民を許可

*フランスとの条約*：
- 賠償金を800万両に増額
- カトリック教会財産の返還
- フランス宣教師が中国全土で活動可能

*ロシアとの条約*：
- ウスリー地域（沿海州）を割譲
- 40万平方マイル以上の領土
- ロシアが戦わずに獲得

### 結果

**即座の影響**：
- 中国が外国外交官の駐在を受け入れることを強制される
- アヘン貿易の合法化
- キリスト教宣教師の保護
- ロシアへの大規模な領土喪失
- 主権のさらなる侵食

**長期的影響**：
- 中国の屈辱を深める
- 抵抗の無益さを実証
- 他の列強が譲歩を求めることを奨励
- 清朝の正統性を弱める
- 太平天国の乱の継続に寄与
- 洋務運動を引き起こす

**文化的影響**：
- 円明園の破壊が屈辱の象徴となる
- 文化的宝物の喪失（多くが今も西洋の博物館に）
- 中国エリートの心理的トラウマ
- 西洋の野蛮性の認識を強化

## 歴史的意義

### 軍事的教訓

**中国の弱点の露呈**：
- 技術的後進性
- 貧弱な軍事組織
- 指揮構造の腐敗
- 不十分な兵站
- 時代遅れの戦術と戦略

**近代化の必要性**：
- 西洋技術の必要性の認識
- 洋務運動の始まり
- 軍隊の近代化の試み
- 体系的問題により成功は限定的

### 外交革命

**朝貢体制の終焉**：
- 伝統的な中国の世界観が挑戦される
- ウェストファリア国家体制の受け入れを強制される
- 対等な外交関係
- 国際法体系への統合

**条約港制度**：
- 中国都市内の外国飛び地
- 治外法権
- 関税自主権の喪失
- 半植民地的地位

### 経済的影響

**アヘン危機**：
- 大量のアヘン輸入が継続
- 銀の流出が加速
- 中毒による社会的荒廃
- 経済混乱

**貿易不均衡**：
- 外国製品が中国市場に氾濫
- 伝統産業が損なわれる
- 外国列強への経済的依存
- 経済主権の喪失

### 社会的・政治的影響

**清朝の正統性危機**：
- 人民を保護できない
- 外国侵略に抵抗できない
- 天命の喪失
- 革命運動の台頭

**知的反応**：
- 伝統的価値観への疑問
- 西洋学習への関心
- 近代化をめぐる議論
- 改革運動の始まり

## 史学的論争

### イギリスの視点（歴史的）
- 中国の貿易妨害への正当な対応
- 条約権利の執行
- 中国を「文明化された」国際社会に導く
- 大英帝国の経済的必要性

### 中国の視点
- 露骨な侵略と帝国主義
- 外国列強による麻薬密売
- 中国主権の侵害
- 国家的屈辱の始まり

### 現代の学術的見解
- 要因の複雑な相互作用
- 双方が責任を共有
- 両立不可能なシステムの衝突
- 誤解と紛争の悲劇
- アヘン貿易は道徳的に弁護できない
- 中国の抵抗は理解できるが無益

---

## 遺産

アヘン戦争は中国の歴史的記憶において深く重要であり続けています：

1. **屈辱の象徴**：中国の衰退の始まりを表す
2. **反帝国主義**：現代中国ナショナリズムの基礎
3. **近代化論争**：伝統対近代化の継続的議論を引き起こす
4. **外交関係**：主権と国際法に対する中国のアプローチに影響
5. **歴史教育**：中国の愛国教育の中心
6. **文化的記憶**：円明園遺跡が思い出として保存される

これらの戦争は、近代世界における技術的・組織的後進性の結果を示し、中国を1世紀以上続く適応と革命の苦しい道に導きました。
